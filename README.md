# 🌿 Home Plants Bot

Бот для догляду за домашніми рослинами.

## 🚀 Функціонал
- Нагадування про полив
- Додавання рослин
- Перегляд списку рослин

## ⚙️ Запуск

1. Клонувати репозиторій:
git clone https://github.com/твій-нік/home-plants-bot

2. Встановити залежності:
pip install -r requirements.txt

3. Створити .env файл:
TOKEN=your_telegram_bot_token

4. Запустити:
python bot.py

## 🗂 Структура
- bot.py — основний файл
- handlers/ — обробники команд
- database/ — робота з БД
